# md文件图片处理程序

把md文件中的图片路径调整为"./.mdname/"的样式，例如："./hello.md"中的图片在"./.hello/"文件夹中。

# 说明

使用AI生成的python代码，pyinstaller 打包为exe，图标AI生成。

## 应用生成命令

```txt
使用python开发一款markdown文件图片处理的app提供以下能力： 
 1、tkinter开发的gui界面 
 2、用户选择指定输入文件夹，程序自动查找这个文件夹下的所有md文件 
 3、用户指定输出文件夹，可以选择按照原来的结构输出或者拍扁文件到平级文件夹 
 4、用户选择按照原来的结构输出就保持md文件与文件夹的结构 
 5、提取每个md文件中使用的图片，把图片更改到与md文件同级的.mdname下的文件夹中，例如：hello.md中的图片应该调整到.hello文件夹下 
 6、注意1.要更改md文件中图片的引用为新的路径；2.把新的md文件要复制到输出文件夹，保持原来的md文件与图片不变；3.图片和md可能重名需要自动重命名；4.注意权限问题，程序在window上运行
```
